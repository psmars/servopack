Create a virtual environment:

- python -m venv venv

Activate the environment:

- source venv/bin/activate

Install the utilities:

- pip install -e .

Go to the directory with the data:

- cd /where/is/the/data

Use one the utility:

- transfer filename
- run
- python
	- from servo import *
	- help()
